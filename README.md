## üîß Installation & Setup

### üìÇ Option 1: Direct Browser Opening (Quick Start)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/dub-x/GameController_landing.git
   cd GameController_landing
   ```

2. **Open in browser:**
   - Simply double-click `index.html` to open it directly in your default browser
   - The page will render immediately without any build process

### üí° Option 2: Using a Local HTTP Server (Recommended)

Using a local server ensures better compatibility with relative paths and modern browser features.

#### Using Node.js (if Node is installed):

```bash
npx http-server
```

Then open the URL shown in your terminal (usually `http://localhost:8080`)

#### Using VS Code Live Server Extension:

1. Install the "Live Server" extension in VS Code
2. Right-click `index.html`
3. Select "Open with Live Server"
4. Your browser will automatically open the page
